Help

frequently asked questions
User Capabilities
  • Who can add, update and delete sports?
    Only organizers can add, update and delete sports.
  • Who can pick the sports to be included in a game?
    Only organizers the organizers who created the game can pick the sports to be included in it.
  • Who can delete a game?
    Only organizers can delete a game.
  • Who can add users/teams to a particular sport or game?
    Only organizers can add users/teams to a particular sport or game.
  • Who can join a sport in a game?
    Competitors can join any sport in a game.
  • Who can view the list of all games?
    Any user (competitor, admin, organizer) including guests can view the list of all games.
  • How do I beome a coach?
    A competitor can become a coach, they will be able to create a team but are not able to create a game.
  • What can the guest view in the application?
    Guest users may only access the web application with minimal usage. They are only allowed to do the following:
    ● View updates through the web application such as events and winners.
  • Who can view and manage my profile?
    Any user can view your profile but only you can manage it.
  • How do I organize an event?
    Organizers can make an event through sending an email to malicsi@up.edu.ph.
  • Who can deactivate and activate accounts?
    Only the administrators are allowed to activate and deactivate accounts, they can also view the system’s logs.
Registration
  • Who can register to Malicsi?
    Only competitors can register to Malicsi.
  • How to register as user in the web application?
    An aspiring competitor can register anytime upon an email request from the admin to get the credentials.
  • Is UP email required to register?
    A UP email is required to register.
  • Do I need to be a registered user to play?
    The Web application will not know whether you are registered or not, but this will be checked in person by the organizers of the game in the event itself.
Technical
  • Do I need internet connection to access the web application?
    Yes, an internet access is needed to access the application.
  • What browser does it work on?
    The application will work on any browser but it is advisable to use Google Chrome.
